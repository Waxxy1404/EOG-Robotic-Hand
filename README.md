# 🤖 EOG-Controlled Robotic Hand

An embedded systems and biomedical engineering project that controls a **5-finger robotic hand** using **EOG (Electrooculography)** signals — electrical potentials generated by eye movements. The signal is captured via an **instrumentation amplifier (INA)**, conditioned through analog filters, digitized by an Arduino ADC, and mapped to specific hand gestures in real time.

> Developed as part of the **Medical Equipments and Instrumentation (EMH 1)** course at **UFPE**, under Prof. Marilú Gomes.

---

## 📋 Overview

Eye movements generate small electrical potentials measurable on the skin around the eyes. This system captures those signals through an analog front-end (INA + filters), reads the processed value via the Arduino's ADC, and maps signal thresholds to distinct robotic hand gestures — each finger independently controlled by a servo motor.

The project was developed incrementally across **4 sketches**, from basic I/O to full EOG-controlled gesture recognition.

## 🗺️ Development Stages

### Sketch 1 — LED Sequencing & Servo Motor Control
- Sequential LED activation via **direct register manipulation** (`DDRB`, `PORTB`)
- Smooth servo rotation from 0° → 180° → 0° using PWM (pin D10)
- Demonstrates low-level hardware control without Arduino abstractions

### Sketch 2 — Serial Communication Test (9600 bps)
- Bidirectional serial communication between Arduino and PC
- Uses `Serial.available()` and `Serial.read()` to echo data back
- Validates baud rate synchronization (data loss occurs if mismatched)

### Sketch 3 — EOG Signal Acquisition & ADC Characterization
- Sinusoidal signal (1 Vpp, 500 mV offset, 1–10 Hz) applied to A0 to simulate EOG
- ADC conversion time measured using `micros()`: **112 µs** → ~8.9 kHz sampling rate
- Validates ADC behavior against theoretical 9.6 kHz (prescaler, 125 kHz clock, 13 clock cycles)

### CODE — Full EOG-Controlled Robotic Hand *(main sketch)*
- Real EOG signal from the analog front-end fed into A0
- Threshold-based gesture detection maps eye movements to hand positions
- 5 servo motors controlled independently for finger-level precision

## ✨ Features

- 👁️ EOG bioelectric signal acquisition via INA + analog filters
- ✋ Independent control of 5 servo motors (one per finger)
- 🤙 Two gestures triggered by signal thresholds:
  - **"OK"** — eyes look **up** (signal > 180): thumb + index close
  - **"Rock n Roll"** — eyes look **down** (signal < 5): middle + ring fingers close
- 😐 Resting state (signal ≈ 60): hand fully open
- 📊 Real-time signal monitoring via Serial Monitor (9600 bps)
- ⚡ 5ms loop for fast response

## 🛠️ Hardware

| Component | Description |
|---|---|
| Arduino Uno | Main microcontroller |
| INA (Instrumentation Amplifier) | Amplifies low-voltage EOG signal from electrodes |
| Analog Filters | Noise conditioning for the bioelectric signal |
| Skin Electrodes | Placed around the eyes to capture EOG signal |
| 5× Servo Motors | One per finger — independent position control |
| Robotic Hand | Mechanical finger structure driven by servos |

## 📌 Pin Configuration

| Pin | Function |
|---|---|
| A0 | EOG signal input (from analog front-end) |
| D3 | Servo 1 — Finger 1 (thumb) |
| D5 | Servo 2 — Finger 2 (index) |
| D6 | Servo 3 — Finger 3 (middle) |
| D9 | Servo 4 — Finger 4 (ring) |
| D10 | Servo 5 — Finger 5 (pinky) |

## 🤙 Gesture Logic

| Eye Movement | EOG Signal | Gesture | Finger Positions |
|---|---|---|---|
| Looking **up** | `> 180` | ✌️ **OK** | 1,2,3 open · 4 closed · 5 open |
| Looking **down** | `< 5` | 🤙 **Rock n Roll** | 1 open · 2,3 closed · 4 open · 5 closed |
| **Resting** | `≈ 60` | ✋ **Open hand** | All fingers open (150°) |

| State | Servo Angle |
|---|---|
| Open | 150° |
| Closed | 30° |

After each gesture, a **3-second delay** holds the position before returning to monitoring.

## 🚀 Getting Started

### Prerequisites

- [Arduino IDE](https://www.arduino.cc/en/software)
- Library: `Servo.h` (built-in with Arduino IDE)

### Installation

```bash
git clone https://github.com/Waxxy1404/eog-robotic-hand.git
```

Open `CODE_fixed.ino` in Arduino IDE.

### Upload

1. Connect Arduino via USB
2. Select correct **Board** (`Arduino Uno`) and **Port**
3. Click **Upload**
4. Open **Serial Monitor** at `9600 baud` to view live EOG values
